# BTL Odoo - Quản lý Khách hàng & Công việc

## Modules
- **nhan_su**: Quản lý nhân viên, phòng ban
- **quan_ly_khach_hang**: Quản lý khách hàng, phản hồi, chiến dịch
- **quan_ly_cong_viec**: Quản lý công việc, tự động gán nhân viên
- **chatbot_noiquy**: Chatbot AI hỏi đáp (Groq API)

## Tính năng nổi bật
- Tự động tạo công việc khi có phản hồi khách hàng
- Chatbot AI trả lời câu hỏi nội quy công ty
